# eth-miner
A Powercord Plugin to mine Eth with.

Balance currently only works on 2miner pools (experimental)

# Instructions

1. Download the plugin
2. Navigate to the Ethereum Miner settings
3. Scroll down
4. Click "Download" next to the T-Rex entry
